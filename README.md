# 🎓 Campus Utility Hub

A modern and interactive **Campus Utility Hub** built using **HTML, CSS, and JavaScript** that provides students with multiple useful campus-related tools and utilities in one place.

🚀 Designed to simplify student life by offering easy access to daily campus utilities through a clean and responsive interface.

---

## 🌟 Features

✨ Responsive User Interface  
✨ Modern and Clean Design  
✨ Multiple Student Utilities in One Platform  
✨ Fast and Lightweight  
✨ Interactive JavaScript Functionalities  
✨ Mobile Friendly Layout  
✨ Easy Navigation  
✨ Beginner Friendly Project Structure  

---
